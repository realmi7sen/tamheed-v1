## Questions

### Question 1
(2+3+3)

a) If $F(x) = \int_{2x}^{x^3} \sqrt{1 + t^4}\, dt$, find $F'(x)$.

b) Compute the integral $\displaystyle\int \frac{dx}{x^{\frac{1}{3}}\left(2 + 3x^{\frac{2}{3}}\right)^4}$.

c) Find the number $c$ in the mean value theorem for $f(x) = \sqrt{1 + 4x}$ on $[0, 2]$.

### Question 2
(3+2+3)

a) Evaluate the integral $\displaystyle\int \frac{(2 + e^{4x})\, dx}{8x + e^{4x}}$.

b) If $y = \dfrac{(3x + 1)^5 \cdot \sqrt[3]{x^3 + 1}}{(1 + x^2)^4 e^x}$, find $\dfrac{dy}{dx}$.

c) Find $\displaystyle\int \frac{e^x \sin^{-1}(e^x)}{\sqrt{1 - e^{2x}}}\, dx$.

### Question 3
(3+3+3)

a) Compute the integral $\displaystyle\int \frac{dx}{x\sqrt{x^8 - 16}}$.

b) Evaluate $\displaystyle\int \frac{x^2\, dx}{\sqrt{4 + x^6}}$.

c) Find $\displaystyle\int \frac{\tan x\, dx}{\sqrt{9 - (\cos x)^2}}$.
