# Midterm 1 Exam — MATH 106 — 1440 H, Semester 2

## Questions

### Question 1
(2+3+3)

a) Find the value of $F'(0)$ if $F(x) = (\cos x)\cdot\displaystyle\int_{0}^{\tan x} \sqrt{1 + t^2}\, dt$.

b) Evaluate the indefinite integral $\displaystyle\int \frac{dx}{\sqrt{x}\,(1 + \sqrt{x})^2}$.

c) Approximate the integral $\displaystyle\int_{0}^{5} \frac{2^x\, dx}{1 + x}$ using Trapezoid rule with $n = 5$.

### Question 2
(3+2+3)

a) Evaluate the integral $\displaystyle\int (\ln x + 1)\, 3^{x\ln x}\, dx$.

b) If $y = \dfrac{x(x^2 + 1)^3}{\sqrt[4]{2x - 1}}$, find $y'$.

c) Compute $\displaystyle\int \frac{\sec^2 x\, dx}{\sqrt{9 - (\tan x)^2}}$.

### Question 3
(3+3+3)

a) Find $\displaystyle\int \frac{dx}{x\sqrt{16x^4 - 1}}$.

b) Evaluate the integral $\displaystyle\int \frac{x^2\, dx}{\sqrt{x^6 - 25}}$.

c) Compute $\displaystyle\int \frac{dx}{x\ln x\sqrt{1 - (\ln x)^4}}$.
