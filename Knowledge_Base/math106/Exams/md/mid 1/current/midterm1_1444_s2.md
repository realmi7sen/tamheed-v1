## Questions

### Question 1
(2+3+3+2+3+3)

1. Let $F(x) = \int_{2x}^{x^2} \frac{dt}{1 + t^4}$. Find $F'(x)$.

2. Use Trapezoid rule with $n = 4$ to approximate the integral $\displaystyle\int_{0}^{4} \frac{dx}{\sqrt{1 + x^3}}$.

3. Find the number $c$ that satisfies the conclusion of the mean value theorem for the function $f(x) = \sqrt{x + 3}$ on $[-2, 6]$.

4. Evaluate the integral $\displaystyle\int \frac{x^2 e^{4x^3}}{1 + e^{4x^3}}\, dx$.

5. Use logarithmic differentiation to find $\dfrac{dy}{dx}$ if $y = x^{2x}\dfrac{(1 + x^3)^{4/3}}{(3 + x^2)^{3/2}}$.

6. Compute the integral $\displaystyle\int \frac{x\, 5^{x^2}}{1 + 5^{2x^2}}\, dx$.

### Question 2
(3+3+3+3+2)

1. Compute $\displaystyle\int \frac{dx}{x\ln x\sqrt{(\ln x)^4 - 1}}$, $\ (x > e)$.

2. Find the indefinite integral $\displaystyle\int \frac{dx}{x\sqrt{4 - x^4}}$, $\ (0 < x < \sqrt{2})$.

3. Evaluate the integral $\displaystyle\int \left(1 + \frac{1}{x}\right)\ln x\, dx$.

4. Find $\displaystyle\int (\tan x)^4 (\sec x)^4\, dx$.

5. Compute the integral $\displaystyle\int \cos(6x)\cos(2x)\, dx$.
