## Questions

### Question 1
(2+3+3)

a) If $F(x) = \int_{x^4}^{2x^6} \sqrt{1 + e^t}\, dt$, find $F'(x)$.

b) Compute the integral $\displaystyle\int \frac{(10x + 15\cosh x)\, dx}{(x^2 + 3\sinh x)^6}$.

c) Use Simpson's rule with $n = 4$ to approximate $\displaystyle\int_{1}^{5} \frac{1}{\sqrt{1 + x^3}}\, dx$.

### Question 2
(3+2+3)

a) Find the number $c$ in the mean value theorem for $f(x) = 3\sqrt{x + 1}$ on $[-1, 8]$.

b) If $y = (2 + x^4)^{x^2 + 3x}$, find $\dfrac{dy}{dx}$.

c) Evaluate the integral $\displaystyle\int \frac{(1 + 2x)e^{2x}}{10 + x^2 e^{4x}}\, dx$.

### Question 3
(3+3+3)

a) Compute the integral $\displaystyle\int \frac{\ln x + 1}{\sqrt{9 - (x\ln x)^2}}\, dx$.

b) Evaluate $\displaystyle\int \frac{dx}{x\sqrt{7 - x^8}}$.

c) Find $\displaystyle\int \frac{\sin x\, dx}{\sec x \sqrt{16(\sin x)^4 - 1}}$.
