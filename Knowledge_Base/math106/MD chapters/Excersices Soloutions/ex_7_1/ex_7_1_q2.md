## Problem

Evaluate $\displaystyle\int x \sin x \, dx$.

---

## Rule Used

**Integration by Parts (7.1):**

$$\int u \, dv = uv - \int v \, du$$

---

## Solution

**Step 1 — Choose $dv$ and $u$.**

The most complicated part of the integrand that can be readily integrated is $\sin x\,dx$. Thus, we let

$$dv = \sin x \, dx \qquad u = x$$

$$v = -\cos x \qquad du = dx$$

**Step 2 — Apply the integration by parts formula.**

$$\int x \sin x \, dx = x(-\cos x) - \int (-\cos x) \, dx$$

$$= -x \cos x + \int \cos x \, dx$$

**Step 3 — Evaluate the remaining integral.**

$$\int \cos x \, dx = \sin x + C$$

**Step 4 — Combine.**

$$\int x \sin x \, dx = -x \cos x + \sin x + C$$

---

## Answer

$$\int x \sin x \, dx = -x \cos x + \sin x + C$$

---

## Check by Differentiation

$$\frac{d}{dx}\left[-x \cos x + \sin x\right] = -\cos x + x \sin x + \cos x = x \sin x \checkmark$$
