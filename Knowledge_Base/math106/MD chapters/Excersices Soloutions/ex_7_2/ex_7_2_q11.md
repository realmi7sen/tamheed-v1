## Problem
Evaluate $\displaystyle\int \tan^3 x \sec^3 x \, dx$.

## Rule Used
**Guidelines (7.3), case 1 (odd power of tangent):** split off $\sec x\tan x$, convert using $\tan^2 x = \sec^2 x - 1$, substitute $u = \sec x$.

## Solution
**Step 1 — Split off the factor $\sec x\tan x$.**
$$\int \tan^3 x\sec^3 x \, dx = \int \tan^2 x\sec^2 x(\sec x\tan x) \, dx = \int (\sec^2 x - 1)\sec^2 x(\sec x\tan x) \, dx$$

**Step 2 — Substitute** $u = \sec x$, $du = \sec x\tan x \, dx$:
$$= \int (u^2 - 1)u^2 \, du = \int (u^4 - u^2) \, du = \frac{u^5}{5} - \frac{u^3}{3} + C$$

**Step 3 — Return to $x$.**

## Answer
$$\int \tan^3 x \sec^3 x \, dx = \frac{1}{5}\sec^5 x - \frac{1}{3}\sec^3 x + C$$

## Check by Differentiation
$$\frac{d}{dx}[\,\cdot\,] = \sec^4 x(\sec x\tan x) - \sec^2 x(\sec x\tan x) = \sec^3 x\tan x(\sec^2 x - 1) = \tan^3 x\sec^3 x \checkmark$$
