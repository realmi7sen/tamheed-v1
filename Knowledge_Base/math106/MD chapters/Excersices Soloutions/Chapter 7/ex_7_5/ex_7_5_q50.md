## Problem
Use Theorem (7.6) to evaluate $\displaystyle\int \frac{1}{\tan x + \sin x} \, dx$.

## Rule Used
**Theorem (7.6) — half-angle substitution** $u = \tan\frac{x}{2}$, after writing $\tan x = \dfrac{\sin x}{\cos x}$.

## Solution
**Step 1 — Rewrite the denominator.**
$$\tan x + \sin x = \frac{\sin x}{\cos x} + \sin x = \frac{\sin x(1 + \cos x)}{\cos x}$$

**Step 2 — Express each piece via Theorem (7.6).**
$$\sin x = \frac{2u}{1 + u^2}, \qquad \cos x = \frac{1 - u^2}{1 + u^2}, \qquad 1 + \cos x = \frac{2}{1 + u^2}$$
$$\tan x + \sin x = \frac{\dfrac{2u}{1 + u^2}\cdot\dfrac{2}{1 + u^2}}{\dfrac{1 - u^2}{1 + u^2}} = \frac{4u}{(1 + u^2)(1 - u^2)}$$

**Step 3 — Substitute with** $dx = \dfrac{2}{1 + u^2}du$:
$$\int \frac{(1 + u^2)(1 - u^2)}{4u}\cdot\frac{2}{1 + u^2} \, du = \int \frac{1 - u^2}{2u} \, du = \int \left(\frac{1}{2u} - \frac{u}{2}\right) du$$

**Step 4 — Integrate.**
$$= \frac{1}{2}\ln|u| - \frac{u^2}{4} + C$$

## Answer
$$\int \frac{1}{\tan x + \sin x} \, dx = \frac{1}{2}\ln\left|\tan\frac{x}{2}\right| - \frac{1}{4}\tan^2\frac{x}{2} + C$$
