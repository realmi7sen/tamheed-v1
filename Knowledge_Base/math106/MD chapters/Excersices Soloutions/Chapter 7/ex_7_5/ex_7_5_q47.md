## Problem
Use Theorem (7.6) to evaluate $\displaystyle\int \frac{1}{2 + \sin x} \, dx$.

## Rule Used
**Theorem (7.6) — half-angle substitution** $u = \tan\frac{x}{2}$:
$$\sin x = \frac{2u}{1 + u^2}, \qquad dx = \frac{2}{1 + u^2}\,du$$
followed by **completing the square**.

## Solution
**Step 1 — Apply Theorem (7.6).**
$$\int \frac{1}{2 + \dfrac{2u}{1 + u^2}}\cdot\frac{2}{1 + u^2} \, du = \int \frac{2}{2(1 + u^2) + 2u} \, du = \int \frac{1}{u^2 + u + 1} \, du$$

**Step 2 — Complete the square.**
$$u^2 + u + 1 = \left(u + \frac{1}{2}\right)^2 + \frac{3}{4}$$

**Step 3 — Standard inverse tangent form** with $a = \frac{\sqrt{3}}{2}$:
$$\int \frac{1}{\left(u + \frac{1}{2}\right)^2 + \frac{3}{4}} \, du = \frac{2}{\sqrt{3}}\tan^{-1}\frac{u + \frac{1}{2}}{\frac{\sqrt{3}}{2}} + C = \frac{2}{\sqrt{3}}\tan^{-1}\frac{2u + 1}{\sqrt{3}} + C$$

## Answer
$$\int \frac{1}{2 + \sin x} \, dx = \frac{2}{\sqrt{3}}\tan^{-1}\left(\frac{2\tan\frac{x}{2} + 1}{\sqrt{3}}\right) + C$$
