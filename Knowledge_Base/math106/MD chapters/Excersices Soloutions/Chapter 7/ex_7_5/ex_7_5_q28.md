## Problem
Evaluate $\displaystyle\int \frac{1}{\sqrt[3]{x} + \sqrt[4]{x}} \, dx$.

## Rule Used
**LCD substitution:** roots of index 3 and 4, so $u = x^{1/12}$ where $12 = \operatorname{lcm}(3, 4)$, then **long division**.

## Solution
**Step 1 — Substitute.** Let $u = x^{1/12}$, $x = u^{12}$, $dx = 12u^{11}\,du$; then $\sqrt[3]{x} = u^4$, $\sqrt[4]{x} = u^3$:
$$\int \frac{12u^{11}}{u^4 + u^3} \, du = 12\int \frac{u^{11}}{u^3(u + 1)} \, du = 12\int \frac{u^8}{u + 1} \, du$$

**Step 2 — Long division.**
$$\frac{u^8}{u + 1} = u^7 - u^6 + u^5 - u^4 + u^3 - u^2 + u - 1 + \frac{1}{u + 1}$$

**Step 3 — Integrate.**
$$12\left[\frac{u^8}{8} - \frac{u^7}{7} + \frac{u^6}{6} - \frac{u^5}{5} + \frac{u^4}{4} - \frac{u^3}{3} + \frac{u^2}{2} - u + \ln|u + 1|\right] + C$$

$$= \frac{3u^8}{2} - \frac{12u^7}{7} + 2u^6 - \frac{12u^5}{5} + 3u^4 - 4u^3 + 6u^2 - 12u + 12\ln(u + 1) + C$$

**Step 4 — Return to $x$** with $u = x^{1/12}$ (so $u^8 = x^{2/3}$, $u^6 = x^{1/2}$, $u^4 = x^{1/3}$, $u^3 = x^{1/4}$, etc.).

## Answer
$$\frac{3}{2}x^{2/3} - \frac{12}{7}x^{7/12} + 2x^{1/2} - \frac{12}{5}x^{5/12} + 3x^{1/3} - 4x^{1/4} + 6x^{1/6} - 12x^{1/12} + 12\ln\left(x^{1/12} + 1\right) + C$$
