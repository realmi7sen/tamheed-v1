## Problem
Evaluate $\displaystyle\int \frac{1}{(x^2 - 1)^{3/2}} \, dx$.

## Rule Used
**Trigonometric substitution (7.4)** for the form $(x^2 - a^2)^r$ with $a = 1$, $r = \tfrac{3}{2}$: use $x = \sec\theta$.

## Solution
**Step 1 — Substitute.** Let $x = \sec\theta$, $dx = \sec\theta\tan\theta \, d\theta$. Then
$$(x^2 - 1)^{3/2} = (\sec^2\theta - 1)^{3/2} = (\tan^2\theta)^{3/2} = \tan^3\theta$$

**Step 2 — Rewrite the integral.**
$$\int \frac{\sec\theta\tan\theta}{\tan^3\theta} \, d\theta = \int \frac{\sec\theta}{\tan^2\theta} \, d\theta = \int \frac{1}{\cos\theta}\cdot\frac{\cos^2\theta}{\sin^2\theta} \, d\theta = \int \frac{\cos\theta}{\sin^2\theta} \, d\theta$$

**Step 3 — Integrate** with $w = \sin\theta$:
$$= \int w^{-2} \, dw = -\frac{1}{w} + C = -\csc\theta + C$$

**Step 4 — Return to $x$** (hypotenuse $x$, adjacent $1$, opposite $\sqrt{x^2 - 1}$): $\csc\theta = \dfrac{x}{\sqrt{x^2 - 1}}$

## Answer
$$\int \frac{1}{(x^2 - 1)^{3/2}} \, dx = -\frac{x}{\sqrt{x^2 - 1}} + C$$

## Check by Differentiation
$$\frac{d}{dx}\left[-x(x^2-1)^{-1/2}\right] = -(x^2-1)^{-1/2} + x^2(x^2-1)^{-3/2} = \frac{-(x^2-1) + x^2}{(x^2-1)^{3/2}} = \frac{1}{(x^2-1)^{3/2}} \checkmark$$
