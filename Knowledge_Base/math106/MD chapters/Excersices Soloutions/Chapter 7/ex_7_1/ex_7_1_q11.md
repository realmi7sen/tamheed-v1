## Problem

Evaluate $\displaystyle\int \tan^{-1} x \, dx$.

---

## Rule Used

**Integration by Parts (7.1):**

$$\int u \, dv = uv - \int v \, du$$

with the substitution technique for the resulting integral.

---

## Solution

**Step 1 — Choose $dv$ and $u$.**

The integrand has only one factor, so we let $dv = dx$ (as in the book's Example 3 for $\int \ln x \, dx$):

$$dv = dx \qquad u = \tan^{-1} x$$

$$v = x \qquad du = \frac{1}{1 + x^2} \, dx$$

**Step 2 — Apply the integration by parts formula.**

$$\int \tan^{-1} x \, dx = x \tan^{-1} x - \int \frac{x}{1 + x^2} \, dx$$

**Step 3 — Evaluate the remaining integral by substitution.**

Let $w = 1 + x^2$, so $dw = 2x \, dx$, i.e. $x \, dx = \tfrac{1}{2} dw$:

$$\int \frac{x}{1 + x^2} \, dx = \frac{1}{2}\int \frac{1}{w} \, dw = \frac{1}{2}\ln|w| = \frac{1}{2}\ln(1 + x^2)$$

(The absolute value is unnecessary since $1 + x^2 > 0$.)

**Step 4 — Combine.**

$$\int \tan^{-1} x \, dx = x \tan^{-1} x - \frac{1}{2}\ln(1 + x^2) + C$$

---

## Answer

$$\int \tan^{-1} x \, dx = x \tan^{-1} x - \frac{1}{2}\ln(1 + x^2) + C$$

---

## Check by Differentiation

$$\frac{d}{dx}\left[x \tan^{-1} x - \frac{1}{2}\ln(1 + x^2)\right] = \tan^{-1} x + \frac{x}{1 + x^2} - \frac{x}{1 + x^2} = \tan^{-1} x \checkmark$$
