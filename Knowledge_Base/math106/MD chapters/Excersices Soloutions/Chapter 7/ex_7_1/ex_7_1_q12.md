## Problem

Evaluate $\displaystyle\int \sin^{-1} x \, dx$.

---

## Rule Used

**Integration by Parts (7.1):**

$$\int u \, dv = uv - \int v \, du$$

with the substitution technique for the resulting integral.

---

## Solution

**Step 1 — Choose $dv$ and $u$.**

The integrand has only one factor, so we let

$$dv = dx \qquad u = \sin^{-1} x$$

$$v = x \qquad du = \frac{1}{\sqrt{1 - x^2}} \, dx$$

**Step 2 — Apply the integration by parts formula.**

$$\int \sin^{-1} x \, dx = x \sin^{-1} x - \int \frac{x}{\sqrt{1 - x^2}} \, dx$$

**Step 3 — Evaluate the remaining integral by substitution.**

Let $w = 1 - x^2$, so $dw = -2x \, dx$, i.e. $x \, dx = -\tfrac{1}{2} dw$:

$$\int \frac{x}{\sqrt{1 - x^2}} \, dx = -\frac{1}{2}\int w^{-1/2} \, dw = -\frac{1}{2}\cdot 2w^{1/2} = -\sqrt{1 - x^2}$$

**Step 4 — Combine.**

$$\int \sin^{-1} x \, dx = x \sin^{-1} x - \left(-\sqrt{1 - x^2}\right) + C = x \sin^{-1} x + \sqrt{1 - x^2} + C$$

---

## Answer

$$\int \sin^{-1} x \, dx = x \sin^{-1} x + \sqrt{1 - x^2} + C$$

---

## Check by Differentiation

$$\frac{d}{dx}\left[x \sin^{-1} x + \sqrt{1 - x^2}\right] = \sin^{-1} x + \frac{x}{\sqrt{1 - x^2}} - \frac{x}{\sqrt{1 - x^2}} = \sin^{-1} x \checkmark$$
