## Problem

Evaluate $\displaystyle\int x \sec x \tan x \, dx$.

---

## Rule Used

**Integration by Parts (7.1):**

$$\int u \, dv = uv - \int v \, du$$

together with the standard integral $\displaystyle\int \sec x \, dx = \ln|\sec x + \tan x| + C$.

---

## Solution

**Step 1 — Choose $dv$ and $u$.**

The most complicated part of the integrand that can be readily integrated is $\sec x \tan x \, dx$, since $\frac{d}{dx}\sec x = \sec x \tan x$. Thus, we let

$$dv = \sec x \tan x \, dx \qquad u = x$$

$$v = \sec x \qquad du = dx$$

**Step 2 — Apply the integration by parts formula.**

$$\int x \sec x \tan x \, dx = x \sec x - \int \sec x \, dx$$

**Step 3 — Evaluate the remaining integral.**

$$\int \sec x \, dx = \ln|\sec x + \tan x| + C$$

**Step 4 — Combine.**

$$\int x \sec x \tan x \, dx = x \sec x - \ln|\sec x + \tan x| + C$$

---

## Answer

$$\int x \sec x \tan x \, dx = x \sec x - \ln|\sec x + \tan x| + C$$

---

## Check by Differentiation

$$\frac{d}{dx}\Big[x\sec x - \ln|\sec x + \tan x|\Big] = \sec x + x\sec x\tan x - \frac{\sec x\tan x + \sec^2 x}{\sec x + \tan x}$$

$$= \sec x + x\sec x\tan x - \sec x = x\sec x\tan x \checkmark$$
