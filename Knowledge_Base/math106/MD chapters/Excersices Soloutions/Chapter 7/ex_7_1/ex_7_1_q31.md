## Problem

Evaluate $\displaystyle\int (\ln x)^2 \, dx$.

---

## Rule Used

**Integration by Parts (7.1) applied twice**, using the result of the book's Example 3: $\displaystyle\int \ln x \, dx = x\ln x - x + C$.

---

## Solution

**Step 1 — First integration by parts.**

The integrand has only one factor, so we let

$$dv = dx \qquad u = (\ln x)^2$$

$$v = x \qquad du = 2\ln x \cdot \frac{1}{x} \, dx$$

$$\int (\ln x)^2 \, dx = x(\ln x)^2 - \int x \cdot \frac{2\ln x}{x} \, dx = x(\ln x)^2 - 2\int \ln x \, dx$$

**Step 2 — Second integration by parts (Example 3 of Section 7.1).**

$$dv = dx \qquad u = \ln x$$

$$v = x \qquad du = \frac{1}{x} \, dx$$

$$\int \ln x \, dx = x\ln x - \int dx = x\ln x - x$$

**Step 3 — Combine.**

$$\int (\ln x)^2 \, dx = x(\ln x)^2 - 2(x\ln x - x) + C$$

$$= x(\ln x)^2 - 2x\ln x + 2x + C$$

---

## Answer

$$\int (\ln x)^2 \, dx = x(\ln x)^2 - 2x\ln x + 2x + C$$

---

## Check by Differentiation

$$\frac{d}{dx}\left[x(\ln x)^2 - 2x\ln x + 2x\right] = (\ln x)^2 + 2\ln x - 2\ln x - 2 + 2 = (\ln x)^2 \checkmark$$
