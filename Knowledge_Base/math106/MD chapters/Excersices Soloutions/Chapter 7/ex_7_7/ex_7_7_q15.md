## Problem
Determine whether $\displaystyle\int_{-\infty}^{\infty} x e^{-x^2} \, dx$ converges or diverges; if it converges, find its value.

## Rule Used
**Definition for doubly infinite limits:** split at a convenient point (here $0$) and require **both** pieces to converge:
$$\int_{-\infty}^{\infty} f\,dx = \int_{-\infty}^{0} f\,dx + \int_{0}^{\infty} f\,dx$$

## Solution
**Step 1 — Antiderivative.** With $u = -x^2$, $du = -2x\,dx$:
$$\int x e^{-x^2} \, dx = -\frac{1}{2}e^{-x^2}$$

**Step 2 — Right piece.**
$$\lim_{t \to \infty}\left[-\frac{1}{2}e^{-x^2}\right]_0^t = 0 + \frac{1}{2} = \frac{1}{2}$$

**Step 3 — Left piece.**
$$\lim_{t \to -\infty}\left[-\frac{1}{2}e^{-x^2}\right]_t^0 = -\frac{1}{2} - 0 = -\frac{1}{2}$$

**Step 4 — Both converge, so add.**
$$\frac{1}{2} + \left(-\frac{1}{2}\right) = 0$$

## Answer
The integral **converges** and
$$\int_{-\infty}^{\infty} x e^{-x^2} \, dx = 0$$

## Remark
Writing "$= 0$ because the integrand is odd" **without first showing both halves converge** is not accepted — the definition requires each piece to converge separately. State the rule, then the symmetry.
