## Problem
Evaluate $\displaystyle\int \frac{x^6 - x^3 + 1}{x^4 + 9x^2} \, dx$.

## Rule Used
**Two rules in sequence:** (1) the degree of the numerator exceeds the denominator, so **long division first** — partial fractions apply only to proper rational expressions; (2) then partial fractions with repeated factor $x^2$ and irreducible quadratic $x^2 + 9$.

## Solution
**Step 1 — Long division.** Dividing $x^6 - x^3 + 1$ by $x^4 + 9x^2$:
$$x^6 - x^3 + 1 = (x^4 + 9x^2)(x^2 - 9) + (-x^3 + 81x^2 + 1)$$
$$\frac{x^6 - x^3 + 1}{x^4 + 9x^2} = x^2 - 9 + \frac{-x^3 + 81x^2 + 1}{x^2(x^2 + 9)}$$

**Step 2 — Partial fractions on the proper part.**
$$\frac{-x^3 + 81x^2 + 1}{x^2(x^2 + 9)} = \frac{A}{x} + \frac{B}{x^2} + \frac{Cx + D}{x^2 + 9}$$
$$-x^3 + 81x^2 + 1 = Ax(x^2 + 9) + B(x^2 + 9) + (Cx + D)x^2$$

- $x = 0$: $\;1 = 9B \Rightarrow B = \tfrac{1}{9}$
- Coefficient of $x$: $\;0 = 9A \Rightarrow A = 0$
- Coefficient of $x^3$: $\;-1 = A + C \Rightarrow C = -1$
- Coefficient of $x^2$: $\;81 = B + D \Rightarrow D = 81 - \tfrac{1}{9} = \tfrac{728}{9}$

**Step 3 — Integrate each piece.**
$$\int (x^2 - 9) \, dx = \frac{x^3}{3} - 9x$$
$$\int \frac{1/9}{x^2} \, dx = -\frac{1}{9x}$$
$$\int \frac{-x}{x^2 + 9} \, dx = -\frac{1}{2}\ln(x^2 + 9)$$
$$\int \frac{728/9}{x^2 + 9} \, dx = \frac{728}{9}\cdot\frac{1}{3}\tan^{-1}\frac{x}{3} = \frac{728}{27}\tan^{-1}\frac{x}{3}$$

## Answer
$$\int \frac{x^6 - x^3 + 1}{x^4 + 9x^2} \, dx = \frac{x^3}{3} - 9x - \frac{1}{9x} - \frac{1}{2}\ln(x^2 + 9) + \frac{728}{27}\tan^{-1}\frac{x}{3} + C$$

## Remark
The most common exam error here is skipping the long division. If the numerator's degree $\geq$ denominator's degree and you go straight to partial fractions, the setup is invalid from the first line — exactly the "wrong start" case where later work earns nothing.
