## Problem
Evaluate $\displaystyle\int \frac{2x^2 - 25x - 33}{(x + 1)^2(x - 5)} \, dx$.

## Rule Used
**Partial fraction decomposition — repeated linear factor plus distinct factor:**
$$\frac{2x^2 - 25x - 33}{(x + 1)^2(x - 5)} = \frac{A}{x + 1} + \frac{B}{(x + 1)^2} + \frac{C}{x - 5}$$

## Solution
**Step 1 — Clear denominators.**
$$2x^2 - 25x - 33 = A(x + 1)(x - 5) + B(x - 5) + C(x + 1)^2$$

**Step 2 — Find the constants.**
- $x = -1$: $\;2 + 25 - 33 = -6 = -6B \Rightarrow B = 1$
- $x = 5$: $\;50 - 125 - 33 = -108 = 36C \Rightarrow C = -3$
- Coefficient of $x^2$: $\;2 = A + C \Rightarrow A = 5$

**Step 3 — Integrate.**
$$\int \left(\frac{5}{x + 1} + \frac{1}{(x + 1)^2} - \frac{3}{x - 5}\right) dx$$

## Answer
$$\int \frac{2x^2 - 25x - 33}{(x + 1)^2(x - 5)} \, dx = 5\ln|x + 1| - \frac{1}{x + 1} - 3\ln|x - 5| + C$$
