## Problem
Evaluate $\displaystyle\int \frac{5x - 12}{x(x - 4)} \, dx$.

## Rule Used
**Partial fraction decomposition — distinct linear factors:**
$$\frac{5x - 12}{x(x - 4)} = \frac{A}{x} + \frac{B}{x - 4}$$

## Solution
**Step 1 — Set up and clear denominators.**
$$5x - 12 = A(x - 4) + Bx$$

**Step 2 — Find $A$ and $B$ by choosing convenient values.**
- $x = 0$: $\;-12 = -4A \Rightarrow A = 3$
- $x = 4$: $\;8 = 4B \Rightarrow B = 2$

**Step 3 — Integrate each term.**
$$\int \left(\frac{3}{x} + \frac{2}{x - 4}\right) dx = 3\ln|x| + 2\ln|x - 4| + C$$

## Answer
$$\int \frac{5x - 12}{x(x - 4)} \, dx = 3\ln|x| + 2\ln|x - 4| + C$$

## Check by Differentiation
$$\frac{3}{x} + \frac{2}{x-4} = \frac{3(x-4) + 2x}{x(x-4)} = \frac{5x - 12}{x(x-4)} \checkmark$$
