## Problem
Evaluate $\displaystyle\int \cos^7 x \, dx$.

## Rule Used
**Guidelines (7.2), case 2 (odd power of cosine):** split off one $\cos x$, use $\cos^2 x = 1 - \sin^2 x$, substitute $u = \sin x$.

## Solution
**Step 1 — Split off one factor and convert.**
$$\int \cos^7 x \, dx = \int (\cos^2 x)^3 \cos x \, dx = \int (1 - \sin^2 x)^3 \cos x \, dx$$

**Step 2 — Substitute** $u = \sin x$, $du = \cos x \, dx$, **and expand** $(1-u^2)^3$:
$$(1 - u^2)^3 = 1 - 3u^2 + 3u^4 - u^6$$
$$\int (1 - 3u^2 + 3u^4 - u^6) \, du = u - u^3 + \frac{3}{5}u^5 - \frac{u^7}{7} + C$$

**Step 3 — Return to $x$.**

## Answer
$$\int \cos^7 x \, dx = \sin x - \sin^3 x + \frac{3}{5}\sin^5 x - \frac{1}{7}\sin^7 x + C$$

## Check by Differentiation
$$\frac{d}{dx}[\,\cdot\,] = \cos x\left(1 - 3\sin^2 x + 3\sin^4 x - \sin^6 x\right) = \cos x (1 - \sin^2 x)^3 = \cos^7 x \checkmark$$
