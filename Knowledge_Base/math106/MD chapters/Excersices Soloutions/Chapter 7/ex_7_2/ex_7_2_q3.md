## Problem
Evaluate $\displaystyle\int \sin^2 x \cos^2 x \, dx$.

## Rule Used
**Guidelines (7.2), case 3 (both powers even):** use half-angle formulas. Here it is fastest through the double-angle identity $\sin x\cos x = \tfrac{1}{2}\sin 2x$.

## Solution
**Step 1 — Rewrite with the double-angle identity.**
$$\sin^2 x\cos^2 x = (\sin x\cos x)^2 = \left(\tfrac{1}{2}\sin 2x\right)^2 = \tfrac{1}{4}\sin^2 2x$$

**Step 2 — Apply the half-angle formula** $\sin^2 2x = \dfrac{1 - \cos 4x}{2}$:
$$\int \sin^2 x\cos^2 x \, dx = \frac{1}{4}\int \frac{1 - \cos 4x}{2} \, dx = \frac{1}{8}\int (1 - \cos 4x) \, dx$$

**Step 3 — Integrate.**
$$= \frac{1}{8}\left(x - \frac{\sin 4x}{4}\right) + C$$

## Answer
$$\int \sin^2 x \cos^2 x \, dx = \frac{x}{8} - \frac{\sin 4x}{32} + C$$

## Check by Differentiation
$$\frac{d}{dx}\left[\frac{x}{8} - \frac{\sin 4x}{32}\right] = \frac{1}{8} - \frac{\cos 4x}{8} = \frac{1 - \cos 4x}{8} = \frac{\sin^2 2x}{4} = \sin^2 x\cos^2 x \checkmark$$
