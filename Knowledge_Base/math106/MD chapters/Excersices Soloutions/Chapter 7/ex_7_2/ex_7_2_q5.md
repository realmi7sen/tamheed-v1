## Problem
Evaluate $\displaystyle\int \sin^3 x \cos^2 x \, dx$.

## Rule Used
**Guidelines (7.2), case 1 (odd power of sine):** split off one $\sin x$, use $\sin^2 x = 1 - \cos^2 x$, substitute $u = \cos x$.

## Solution
**Step 1 — Split off one factor of $\sin x$ and convert.**
$$\int \sin^3 x\cos^2 x \, dx = \int (1 - \cos^2 x)\cos^2 x \sin x \, dx$$

**Step 2 — Substitute** $u = \cos x$, $du = -\sin x \, dx$:
$$= -\int (1 - u^2)u^2 \, du = -\int (u^2 - u^4) \, du = -\frac{u^3}{3} + \frac{u^5}{5} + C$$

**Step 3 — Return to $x$.**

## Answer
$$\int \sin^3 x \cos^2 x \, dx = -\frac{1}{3}\cos^3 x + \frac{1}{5}\cos^5 x + C$$

## Check by Differentiation
$$\frac{d}{dx}[\,\cdot\,] = \cos^2 x\sin x - \cos^4 x\sin x = \sin x\cos^2 x(1 - \cos^2 x) = \sin^3 x\cos^2 x \checkmark$$
