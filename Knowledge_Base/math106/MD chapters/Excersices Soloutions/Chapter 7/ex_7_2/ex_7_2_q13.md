## Problem
Evaluate $\displaystyle\int \tan^6 x \, dx$.

## Rule Used
**Repeated use of the identity** $\tan^2 x = \sec^2 x - 1$, peeling off $\tan^2 x$ each time; the pieces with $\sec^2 x$ integrate by $u = \tan x$.

## Solution
**Step 1 — Peel off $\tan^2 x$.**
$$\int \tan^6 x \, dx = \int \tan^4 x(\sec^2 x - 1) \, dx = \int \tan^4 x\sec^2 x \, dx - \int \tan^4 x \, dx$$

**Step 2 — First piece** ($u = \tan x$): $\displaystyle\int \tan^4 x\sec^2 x \, dx = \frac{\tan^5 x}{5}$

**Step 3 — Repeat on $\tan^4 x$.**
$$\int \tan^4 x \, dx = \int \tan^2 x\sec^2 x \, dx - \int \tan^2 x \, dx = \frac{\tan^3 x}{3} - \int (\sec^2 x - 1) \, dx = \frac{\tan^3 x}{3} - \tan x + x$$

**Step 4 — Combine.**
$$\int \tan^6 x \, dx = \frac{\tan^5 x}{5} - \frac{\tan^3 x}{3} + \tan x - x + C$$

## Answer
$$\int \tan^6 x \, dx = \frac{1}{5}\tan^5 x - \frac{1}{3}\tan^3 x + \tan x - x + C$$

## Check by Differentiation
$$\frac{d}{dx}[\,\cdot\,] = \tan^4 x\sec^2 x - \tan^2 x\sec^2 x + \sec^2 x - 1 = \sec^2 x(\tan^4 x - \tan^2 x + 1) - 1$$
Using $\sec^2 x = 1+\tan^2 x$ and expanding gives $\tan^6 x$ $\checkmark$
