## Problem
Evaluate $\displaystyle\int \tan^3 x \sec^4 x \, dx$.

## Rule Used
**Guidelines (7.3), case 2 (even power of secant):** split off $\sec^2 x$, convert the rest using $\sec^2 x = 1 + \tan^2 x$, substitute $u = \tan x$.

## Solution
**Step 1 — Split off $\sec^2 x$ and convert.**
$$\int \tan^3 x\sec^4 x \, dx = \int \tan^3 x(1 + \tan^2 x)\sec^2 x \, dx$$

**Step 2 — Substitute** $u = \tan x$, $du = \sec^2 x \, dx$:
$$= \int u^3(1 + u^2) \, du = \int (u^3 + u^5) \, du = \frac{u^4}{4} + \frac{u^6}{6} + C$$

**Step 3 — Return to $x$.**

## Answer
$$\int \tan^3 x \sec^4 x \, dx = \frac{1}{4}\tan^4 x + \frac{1}{6}\tan^6 x + C$$

## Check by Differentiation
$$\frac{d}{dx}[\,\cdot\,] = \tan^3 x\sec^2 x + \tan^5 x\sec^2 x = \tan^3 x\sec^2 x(1 + \tan^2 x) = \tan^3 x\sec^4 x \checkmark$$
