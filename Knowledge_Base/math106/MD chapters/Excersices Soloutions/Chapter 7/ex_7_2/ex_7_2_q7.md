## Problem
Evaluate $\displaystyle\int \sin^6 x \, dx$.

## Rule Used
**Guidelines (7.2), case 3 (even power):** apply the half-angle formula $\sin^2 x = \dfrac{1 - \cos 2x}{2}$ repeatedly, plus case 2 for the odd power $\cos^3 2x$ that appears.

## Solution
**Step 1 — Cube the half-angle formula.**
$$\sin^6 x = \left(\frac{1 - \cos 2x}{2}\right)^3 = \frac{1}{8}\left(1 - 3\cos 2x + 3\cos^2 2x - \cos^3 2x\right)$$

**Step 2 — Integrate term by term.**

$\displaystyle\int 1 \, dx = x$, $\qquad \displaystyle\int \cos 2x \, dx = \frac{\sin 2x}{2}$

For the even power, half-angle again:
$$\int \cos^2 2x \, dx = \int \frac{1 + \cos 4x}{2} \, dx = \frac{x}{2} + \frac{\sin 4x}{8}$$

For the odd power, case 2 with $u = \sin 2x$:
$$\int \cos^3 2x \, dx = \int (1 - \sin^2 2x)\cos 2x \, dx = \frac{\sin 2x}{2} - \frac{\sin^3 2x}{6}$$

**Step 3 — Combine.**
$$\int \sin^6 x \, dx = \frac{1}{8}\left[x - \frac{3\sin 2x}{2} + \frac{3x}{2} + \frac{3\sin 4x}{8} - \frac{\sin 2x}{2} + \frac{\sin^3 2x}{6}\right] + C$$

$$= \frac{1}{8}\left[\frac{5x}{2} - 2\sin 2x + \frac{3\sin 4x}{8} + \frac{\sin^3 2x}{6}\right] + C$$

## Answer
$$\int \sin^6 x \, dx = \frac{5x}{16} - \frac{\sin 2x}{4} + \frac{3\sin 4x}{64} + \frac{\sin^3 2x}{48} + C$$

## Remark
Using $\sin^3 2x = \tfrac{1}{4}(3\sin 2x - \sin 6x)$, this is equivalent to the tabulated form $\frac{5x}{16} - \frac{15\sin 2x}{64} + \frac{3\sin 4x}{64} - \frac{\sin 6x}{192} + C$. Answers that differ by trigonometric identities are the same function.
