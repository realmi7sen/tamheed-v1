## Problem
Evaluate $\displaystyle\int \cos^3 x \, dx$.

## Rule Used
**Guidelines (7.2), case 2 (odd power of cosine):** split off one $\cos x$, convert the rest using $\cos^2 x = 1 - \sin^2 x$, substitute $u = \sin x$.

## Solution
**Step 1 — Split off one factor of $\cos x$.**
$$\int \cos^3 x \, dx = \int \cos^2 x \cos x \, dx = \int (1 - \sin^2 x)\cos x \, dx$$

**Step 2 — Substitute** $u = \sin x$, $du = \cos x \, dx$:
$$= \int (1 - u^2) \, du = u - \frac{u^3}{3} + C$$

**Step 3 — Return to $x$.**
$$= \sin x - \frac{\sin^3 x}{3} + C$$

## Answer
$$\int \cos^3 x \, dx = \sin x - \frac{1}{3}\sin^3 x + C$$

## Check by Differentiation
$$\frac{d}{dx}\left[\sin x - \tfrac{1}{3}\sin^3 x\right] = \cos x - \sin^2 x\cos x = \cos x(1 - \sin^2 x) = \cos^3 x \checkmark$$
