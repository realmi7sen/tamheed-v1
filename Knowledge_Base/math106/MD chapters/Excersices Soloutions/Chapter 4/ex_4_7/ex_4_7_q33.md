## Problem

Use Simpson's rule, with $n=4$, to approximate the average value of $f$ on the given interval.
$$f(x) = \frac{1}{x^4 + 1}; \quad [0, 4]$$

---

## Rule Used

**Average Value of a Function**
$$f_{av} = \frac{1}{b - a} \int_a^b f(x) \, dx$$

**Simpson's Rule**
$$S_n = \frac{\Delta x}{3} [f(x_0) + 4f(x_1) + 2f(x_2) + \dots + f(x_n)]$$

---

## Solution

**Step 1 — Set up the average value formula**
$$f_{av} = \frac{1}{4 - 0} \int_0^4 \frac{1}{x^4 + 1} \, dx = \frac{1}{4} \int_0^4 \frac{1}{x^4 + 1} \, dx$$
We will use Simpson's rule to approximate the integral part.

**Step 2 — Calculate grid parameters for Simpson's Rule**
Interval $[0, 4]$ with $n = 4$:
$$\Delta x = \frac{4 - 0}{4} = 1$$
The grid points are $x_0=0, x_1=1, x_2=2, x_3=3, x_4=4$.

**Step 3 — Evaluate the function at the grid points**
- $f(0) = \frac{1}{0^4 + 1} = 1$
- $f(1) = \frac{1}{1^4 + 1} = \frac{1}{2} = 0.5$
- $f(2) = \frac{1}{2^4 + 1} = \frac{1}{17} \approx 0.0588$
- $f(3) = \frac{1}{3^4 + 1} = \frac{1}{82} \approx 0.0122$
- $f(4) = \frac{1}{4^4 + 1} = \frac{1}{257} \approx 0.0039$

**Step 4 — Apply Simpson's Rule ($S_4$)**
$$S_4 = \frac{1}{3} \left[ f(0) + 4f(1) + 2f(2) + 4f(3) + f(4) \right]$$
$$S_4 \approx \frac{1}{3} \left[ 1 + 4(0.5) + 2(0.0588) + 4(0.0122) + 0.0039 \right]$$
$$S_4 \approx \frac{1}{3} \left[ 1 + 2 + 0.1176 + 0.0488 + 0.0039 \right]$$
$$S_4 \approx \frac{1}{3} [3.1703] \approx 1.0568$$

**Step 5 — Find the average value**
Multiply the integral approximation by $\frac{1}{4}$:
$$f_{av} \approx \frac{1}{4}(1.0568) \approx 0.2642$$

---

## Answer

$$\approx 0.264$$
