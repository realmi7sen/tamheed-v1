## Problem

(a) Approximate the definite integral using the indicated rule for the given values of $n$. 
(b) On the basis of the pattern of values, determine the expected accuracy for the approximation corresponding to the largest $n$.
$$\int_0^\pi \cos(\sin x) \, dx; \quad \text{Simpson's rule for } n = 2, 6, 18, \text{ and } 54$$

---

## Rule Used

**Simpson's Rule**
$$S_n = \frac{\Delta x}{3} [f(x_0) + 4f(x_1) + 2f(x_2) + 4f(x_3) + \dots + 4f(x_{n-1}) + f(x_n)]$$
*(Requires $n$ to be an even integer).*

**Error Estimate for Simpson's Rule**
The error $E_S$ is inversely proportional to $n^4$ ($E_S \propto \frac{1}{n^4}$).

---

## Solution

**Step 1 — Setup for $n=2$ (Part a)**
Calculate the subinterval width $\Delta x$:
$$\Delta x = \frac{\pi - 0}{2} = \frac{\pi}{2}$$
The grid points are $x_0=0, x_1=\frac{\pi}{2}, x_2=\pi$.
Apply Simpson's Rule formula:
$$S_2 = \frac{\pi/2}{3} \left[ f(0) + 4f\left(\frac{\pi}{2}\right) + f(\pi) \right]$$
Evaluate the function $f(x) = \cos(\sin x)$:
$$S_2 = \frac{\pi}{6} \left[ \cos(\sin 0) + 4\cos\left(\sin \frac{\pi}{2}\right) + \cos(\sin \pi) \right]$$
$$S_2 = \frac{\pi}{6} \left[ \cos(0) + 4\cos(1) + \cos(0) \right] = \frac{\pi}{6} [1 + 4\cos(1) + 1] = \frac{\pi}{6} [2 + 4\cos(1)]$$
*(Repeating via CAS for $n=6, 18, 54$ generates the full approximation sequence).*

**Step 2 — Determine expected accuracy pattern (Part b)**
The sequence of values for $n$ triples each time ($2 \to 6 \to 18 \to 54$).
Because the error for Simpson's Rule is proportional to $\frac{1}{n^4}$, multiplying $n$ by $3$ divides the error by $3^4 = 81$.
Thus, each subsequent approximation is expected to have an error roughly 81 times smaller than the previous one, leading to extreme precision for $n=54$.

---

## Answer

(a) Setup shown for $S_n$.
(b) The values of $n$ increase by a factor of 3. Since error $\propto 1/n^4$, each step reduces the expected error by a factor of 81.
