## Problem

Use Simpson's rule, with $n=4$, to approximate the average value of $f$ on the given interval.
$$f(x) = \sqrt{\cos x}; \quad [-1, 1]$$

---

## Rule Used

**Average Value of a Function**
$$f_{av} = \frac{1}{b - a} \int_a^b f(x) \, dx$$

**Simpson's Rule**
$$S_n = \frac{\Delta x}{3} [f(x_0) + 4f(x_1) + 2f(x_2) + \dots + f(x_n)]$$

---

## Solution

**Step 1 — Set up the average value formula**
$$f_{av} = \frac{1}{1 - (-1)} \int_{-1}^1 \sqrt{\cos x} \, dx = \frac{1}{2} \int_{-1}^1 \sqrt{\cos x} \, dx$$
*(Note: Ensure your calculator is set to radians).*

**Step 2 — Calculate grid parameters for Simpson's Rule**
Interval $[-1, 1]$ with $n = 4$:
$$\Delta x = \frac{1 - (-1)}{4} = \frac{2}{4} = 0.5$$
The grid points are $x_0=-1, x_1=-0.5, x_2=0, x_3=0.5, x_4=1$.

**Step 3 — Evaluate the function at the grid points**
- $f(-1) = \sqrt{\cos(-1)} = \sqrt{\cos(1)} \approx 0.7351$
- $f(-0.5) = \sqrt{\cos(-0.5)} = \sqrt{\cos(0.5)} \approx 0.9368$
- $f(0) = \sqrt{\cos(0)} = \sqrt{1} = 1$
- $f(0.5) = \sqrt{\cos(0.5)} \approx 0.9368$
- $f(1) = \sqrt{\cos(1)} \approx 0.7351$

**Step 4 — Apply Simpson's Rule ($S_4$)**
$$S_4 = \frac{0.5}{3} \left[ f(-1) + 4f(-0.5) + 2f(0) + 4f(0.5) + f(1) \right]$$
$$S_4 \approx \frac{1}{6} \left[ 0.7351 + 4(0.9368) + 2(1) + 4(0.9368) + 0.7351 \right]$$
$$S_4 \approx \frac{1}{6} \left[ 0.7351 + 3.7472 + 2 + 3.7472 + 0.7351 \right]$$
$$S_4 \approx \frac{1}{6} [10.9646] \approx 1.8274$$

**Step 5 — Find the average value**
Multiply the integral approximation by $\frac{1}{2}$:
$$f_{av} \approx \frac{1}{2}(1.8274) \approx 0.9137$$
