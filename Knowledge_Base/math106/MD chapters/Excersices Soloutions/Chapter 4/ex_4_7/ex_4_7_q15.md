## Problem

(a) Approximate the definite integral using the indicated rule for the given values of $n$. 
(b) On the basis of the pattern of values, determine the expected accuracy for the approximation corresponding to the largest $n$.
$$\int_1^3 \sqrt{1 + x^3} \, dx; \quad \text{trapezoidal rule for } n = 5, 10, 20, \text{ and } 40$$

---

## Rule Used

**Trapezoidal Rule**
$$T_n = \frac{\Delta x}{2} [f(x_0) + 2f(x_1) + 2f(x_2) + \dots + 2f(x_{n-1}) + f(x_n)]$$
where $\Delta x = \frac{b - a}{n}$.

**Error Estimate for Trapezoidal Rule**
The error $E_T$ is inversely proportional to $n^2$ ($E_T \propto \frac{1}{n^2}$).

---

## Solution

**Step 1 — Setup for $n=5$ (Part a)**
Calculate the subinterval width $\Delta x$:
$$\Delta x = \frac{3 - 1}{5} = \frac{2}{5} = 0.4$$
The grid points are $x_0=1, x_1=1.4, x_2=1.8, x_3=2.2, x_4=2.6, x_5=3$.
Apply the Trapezoidal Rule formula:
$$T_5 = \frac{0.4}{2} \left[ f(1) + 2f(1.4) + 2f(1.8) + 2f(2.2) + 2f(2.6) + f(3) \right]$$
$$T_5 = 0.2 \left[ \sqrt{2} + 2\sqrt{1.4^3+1} + 2\sqrt{1.8^3+1} + 2\sqrt{2.2^3+1} + 2\sqrt{2.6^3+1} + \sqrt{28} \right]$$
*(Using a calculator, this sequence yields $T_5 \approx 6.2498$. Repeating this automated process for $n=10, 20, 40$ yields increasingly precise values approaching the true area).*

**Step 2 — Determine expected accuracy pattern (Part b)**
The sequence of values for $n$ doubles each time ($5 \to 10 \to 20 \to 40$).
Because the error bound for the Trapezoidal Rule involves $\frac{1}{n^2}$, doubling the number of subintervals $n$ divides the error by $2^2 = 4$.
Therefore, as we move to $n=40$, the error is expected to be reduced by a factor of roughly $4$ with each step, leading to high decimal accuracy (typically accurate to at least 3-4 decimal places for $n=40$ for smooth functions).

---

## Answer

(a) Setup shown for $T_n$.
(b) Because the error for the trapezoidal rule is proportional to $1/n^2$, doubling $n$ reduces the error by a factor of 4.
