## Problem

(a) Approximate the definite integral using the indicated rule for the given values of $n$. 
(b) On the basis of the pattern of values, determine the expected accuracy for the approximation corresponding to the largest $n$.
$$\int_0^5 2^{-x^2} \, dx; \quad \text{trapezoidal rule for } n = 2, 8, 32, \text{ and } 128$$

---

## Rule Used

**Trapezoidal Rule**
$$T_n = \frac{\Delta x}{2} [f(x_0) + 2f(x_1) + \dots + 2f(x_{n-1}) + f(x_n)]$$
where $\Delta x = \frac{b - a}{n}$.

---

## Solution

**Step 1 — Setup for $n=2$ (Part a)**
Calculate the subinterval width $\Delta x$:
$$\Delta x = \frac{5 - 0}{2} = 2.5$$
The grid points are $x_0=0, x_1=2.5, x_2=5$.
Apply the Trapezoidal Rule formula:
$$T_2 = \frac{2.5}{2} \left[ f(0) + 2f(2.5) + f(5) \right]$$
$$T_2 = 1.25 \left[ 2^0 + 2(2^{-(2.5)^2}) + 2^{-25} \right] = 1.25 \left[ 1 + 2(2^{-6.25}) + 2^{-25} \right]$$
*(This process is repeated via calculator for $n=8, 32, 128$ to establish the sequence of values).*

**Step 2 — Determine expected accuracy pattern (Part b)**
The sequence of values for $n$ multiplies by $4$ each time ($2 \to 8 \to 32 \to 128$).
Because the theoretical error bound for the Trapezoidal Rule is proportional to $\frac{1}{n^2}$, multiplying $n$ by $4$ divides the error by $4^2 = 16$.
Therefore, the error is reduced by a factor of 16 at each step, making the approximation for $n=128$ extremely highly accurate.

---

## Answer

(a) Setup shown for $T_n$.
(b) The values of $n$ increase by a factor of 4. Since error $\propto 1/n^2$, each step reduces the expected error by a factor of 16.
