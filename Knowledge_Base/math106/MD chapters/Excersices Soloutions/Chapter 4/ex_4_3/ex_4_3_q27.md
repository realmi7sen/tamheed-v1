## Problem

Let $A$ be the area under the graph of the given function $f$ from $a$ to $b$. Approximate $A$ by dividing $[a, b]$ into subintervals of equal length $\Delta x$ and using (a) $A_{IP}$ (inscribed rectangles) and (b) $A_{CP}$ (circumscribed rectangles).
$$f(x) = 3 - x; \quad a = -2, \quad b = 2; \quad \Delta x = 1$$

---

## Rule Used

**Area Approximations**
- **Inscribed Rectangles ($A_{IP}$):** Use the minimum value of $f(x)$ on each subinterval to define the height.
- **Circumscribed Rectangles ($A_{CP}$):** Use the maximum value of $f(x)$ on each subinterval to define the height.
$$A \approx \sum_{k=1}^n f(x_k^*) \Delta x$$

---

## Solution

**Step 1 — Identify the subintervals and function behavior**
The interval is $[-2, 2]$ with width $\Delta x = 1$. The $4$ subintervals are:
$$[-2, -1], [-1, 0], [0, 1], [1, 2]$$
The function $f(x) = 3 - x$ is a line with a negative slope, meaning it is strictly decreasing. Therefore:
- The **minimum** on any interval $[x_{k-1}, x_k]$ occurs at the right endpoint ($x_k$).
- The **maximum** on any interval $[x_{k-1}, x_k]$ occurs at the left endpoint ($x_{k-1}$).

**Step 2 — Calculate $A_{IP}$ (Inscribed Rectangles)**
Since the function is decreasing, the inscribed (smallest) rectangles are formed using the right endpoints: $x_1 = -1$, $x_2 = 0$, $x_3 = 1$, $x_4 = 2$.
Evaluate the heights:
$$f(-1) = 3 - (-1) = 4$$
$$f(0) = 3 - 0 = 3$$
$$f(1) = 3 - 1 = 2$$
$$f(2) = 3 - 2 = 1$$
Sum the areas (Height $\times \Delta x$):
$$A_{IP} = [f(-1) + f(0) + f(1) + f(2)] \Delta x$$
$$A_{IP} = (4 + 3 + 2 + 1)(1) = 10$$

**Step 3 — Calculate $A_{CP}$ (Circumscribed Rectangles)**
Since the function is decreasing, the circumscribed (largest) rectangles are formed using the left endpoints: $x_0 = -2$, $x_1 = -1$, $x_2 = 0$, $x_3 = 1$.
Evaluate the heights:
$$f(-2) = 3 - (-2) = 5$$
$$f(-1) = 3 - (-1) = 4$$
$$f(0) = 3 - 0 = 3$$
$$f(1) = 3 - 1 = 2$$
Sum the areas:
$$A_{CP} = [f(-2) + f(-1) + f(0) + f(1)] \Delta x$$
$$A_{CP} = (5 + 4 + 3 + 2)(1) = 14$$

---

## Answer

(a) $A_{IP} = 10$
(b) $A_{CP} = 14$
