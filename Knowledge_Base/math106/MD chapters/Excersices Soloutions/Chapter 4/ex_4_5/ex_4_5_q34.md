## Problem

If $f$ is continuous on $[a, b]$, prove that
$$\left| \int_a^b f(x) \, dx \right| \le \int_a^b |f(x)| \, dx$$
*(Hint: $-|f(x)| \le f(x) \le |f(x)|$.)*

---

## Rule Used

**Comparison Theorem for Integrals**
If $f(x) \le g(x)$ for all $x$ in $[a, b]$, then $\int_a^b f(x) \, dx \le \int_a^b g(x) \, dx$.

**Absolute Value Inequality Property**
For any real number $u$ and $c \ge 0$:
$-c \le u \le c \iff |u| \le c$

---

## Solution

**Step 1 — Start with the property of absolute values**
For any real number $y$, it is always true that $-|y| \le y \le |y|$. Applying this to our function $f(x)$ for all $x$ in $[a, b]$, we have:
$$-|f(x)| \le f(x) \le |f(x)|$$

**Step 2 — Apply the Comparison Theorem**
Since $f$ is continuous, $|f(x)|$ is also continuous, and the integrals of all these functions exist on $[a, b]$. We integrate each part of the inequality from $a$ to $b$:
$$\int_a^b -|f(x)| \, dx \le \int_a^b f(x) \, dx \le \int_a^b |f(x)| \, dx$$

**Step 3 — Factor out the constant**
Using the linearity property of integrals, factor the negative sign out of the first integral:
$$-\int_a^b |f(x)| \, dx \le \int_a^b f(x) \, dx \le \int_a^b |f(x)| \, dx$$

**Step 4 — Apply the absolute value property**
Let $u = \int_a^b f(x) \, dx$ and let $c = \int_a^b |f(x)| \, dx$. Since the integral of an absolute value is always non-negative, $c \ge 0$.
The inequality is now in the form $-c \le u \le c$.
By the absolute value inequality property, this is equivalent to $|u| \le c$:
$$\left| \int_a^b f(x) \, dx \right| \le \int_a^b |f(x)| \, dx$$

---

## Answer

The proof follows directly from integrating the absolute value inequality $-|f(x)| \le f(x) \le |f(x)|$ over $[a, b]$ and applying properties of integrals.
