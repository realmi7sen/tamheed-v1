## Problem

The given integral may be verified using the results in Section 4.6. 
(a) Find a number $z$ that satisfies the conclusion of the mean value theorem (4.28). 
(b) Find the average value of $f$ on $[a, b]$.
$$\int_0^3 3x^2 \, dx = 27$$

---

## Rule Used

**Mean Value Theorem for Definite Integrals**
If $f$ is continuous on $[a, b]$, there is a number $z$ in $[a, b]$ such that:
$$\int_a^b f(x) \, dx = f(z)(b - a)$$

**Average Value of a Function**
The average value $f_{av}$ of a function $f$ on $[a, b]$ is:
$$f_{av} = \frac{1}{b - a} \int_a^b f(x) \, dx$$
(Note that $f(z) = f_{av}$).

---

## Solution

**Step 1 — Identify the components**
From the given integral, we have:
$f(x) = 3x^2$
$a = 0, \quad b = 3$
$\int_0^3 3x^2 \, dx = 27$

**Step 2 — Find the average value (Part b)**
Calculate $f_{av}$ using the formula:
$$f_{av} = \frac{1}{3 - 0} (27) = \frac{27}{3} = 9$$

**Step 3 — Find the number $z$ (Part a)**
According to the Mean Value Theorem, we must find a $z$ in $[0, 3]$ such that $f(z) = f_{av}$:
$$3z^2 = 9$$
$$z^2 = 3$$
$$z = \pm\sqrt{3}$$
Since $z$ must be in the interval $[0, 3]$, we reject $-\sqrt{3}$ and keep $z = \sqrt{3}$.

---

## Answer

(a) $z = \sqrt{3}$
(b) $f_{av} = 9$
