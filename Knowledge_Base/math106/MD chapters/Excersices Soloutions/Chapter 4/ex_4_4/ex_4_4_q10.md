## Problem

Find the Riemann sum $R_P$ for the given function $f$ on the indicated partition $P$ by choosing on each subinterval of $P$ (a) the left-hand endpoint, (b) the right-hand endpoint, and (c) the midpoint.
$$f(x) = \sqrt{x}; \quad P = \{1, 3, 4, 9, 12, 16\}, \quad n = 5$$

---

## Rule Used

**Riemann Sum**
$$R_P = \sum_{k=1}^n f(w_k) \Delta x_k$$
where $w_k$ is a sample point in the $k$-th subinterval $[x_{k-1}, x_k]$, and $\Delta x_k = x_k - x_{k-1}$.

---

## Solution

**Step 1 — Determine the subintervals and their lengths ($\Delta x_k$)**
The partition gives five subintervals:
- $[1, 3] \implies \Delta x_1 = 2$
- $[3, 4] \implies \Delta x_2 = 1$
- $[4, 9] \implies \Delta x_3 = 5$
- $[9, 12] \implies \Delta x_4 = 3$
- $[12, 16] \implies \Delta x_5 = 4$

**Step 2 — Calculate $R_P$ using left-hand endpoints (a)**
The left endpoints are $w_1 = 1, w_2 = 3, w_3 = 4, w_4 = 9, w_5 = 12$.
$$R_P = f(1)\Delta x_1 + f(3)\Delta x_2 + f(4)\Delta x_3 + f(9)\Delta x_4 + f(12)\Delta x_5$$
Evaluate the function $f(x) = \sqrt{x}$:
$$R_P = (\sqrt{1})(2) + (\sqrt{3})(1) + (\sqrt{4})(5) + (\sqrt{9})(3) + (\sqrt{12})(4)$$
$$R_P = (1)(2) + (\sqrt{3})(1) + (2)(5) + (3)(3) + (2\sqrt{3})(4)$$
$$R_P = 2 + \sqrt{3} + 10 + 9 + 8\sqrt{3}$$
$$R_P = 21 + 9\sqrt{3}$$

**Step 3 — Calculate $R_P$ using right-hand endpoints (b)**
The right endpoints are $w_1 = 3, w_2 = 4, w_3 = 9, w_4 = 12, w_5 = 16$.
$$R_P = f(3)\Delta x_1 + f(4)\Delta x_2 + f(9)\Delta x_3 + f(12)\Delta x_4 + f(16)\Delta x_5$$
$$R_P = (\sqrt{3})(2) + (\sqrt{4})(1) + (\sqrt{9})(5) + (\sqrt{12})(3) + (\sqrt{16})(4)$$
$$R_P = 2\sqrt{3} + (2)(1) + (3)(5) + (2\sqrt{3})(3) + (4)(4)$$
$$R_P = 2\sqrt{3} + 2 + 15 + 6\sqrt{3} + 16$$
$$R_P = 33 + 8\sqrt{3}$$

**Step 4 — Calculate $R_P$ using midpoints (c)**
The midpoints are $w_1 = 2, w_2 = 3.5 = \frac{7}{2}, w_3 = 6.5 = \frac{13}{2}, w_4 = 10.5 = \frac{21}{2}, w_5 = 14$.
$$R_P = f(2)\Delta x_1 + f(3.5)\Delta x_2 + f(6.5)\Delta x_3 + f(10.5)\Delta x_4 + f(14)\Delta x_5$$
$$R_P = (\sqrt{2})(2) + \left(\sqrt{\frac{7}{2}}\right)(1) + \left(\sqrt{\frac{13}{2}}\right)(5) + \left(\sqrt{\frac{21}{2}}\right)(3) + (\sqrt{14})(4)$$
$$R_P = 2\sqrt{2} + \frac{\sqrt{14}}{2} + \frac{5\sqrt{26}}{2} + \frac{3\sqrt{42}}{2} + 4\sqrt{14}$$
Combine the $\sqrt{14}$ terms: $\frac{1}{2}\sqrt{14} + \frac{8}{2}\sqrt{14} = \frac{9}{2}\sqrt{14}$
$$R_P = 2\sqrt{2} + \frac{9}{2}\sqrt{14} + \frac{5}{2}\sqrt{26} + \frac{3}{2}\sqrt{42}$$

---

## Answer

(a) $21 + 9\sqrt{3}$
(b) $33 + 8\sqrt{3}$
(c) $2\sqrt{2} + \frac{9}{2}\sqrt{14} + \frac{5}{2}\sqrt{26} + \frac{3}{2}\sqrt{42}$
