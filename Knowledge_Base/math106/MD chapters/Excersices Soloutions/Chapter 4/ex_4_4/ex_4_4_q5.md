## Problem

Find the Riemann sum $R_P$ for the given function $f$ on the indicated partition $P$ by choosing on each subinterval of $P$ (a) the left-hand endpoint, (b) the right-hand endpoint, and (c) the midpoint.
$$f(x) = 2x + 3; \quad P = \{1, 3, 4, 5\}, \quad n = 3$$

---

## Rule Used

**Riemann Sum**
$$R_P = \sum_{k=1}^n f(w_k) \Delta x_k$$
where $w_k$ is a sample point in the $k$-th subinterval $[x_{k-1}, x_k]$, and $\Delta x_k = x_k - x_{k-1}$.

---

## Solution

**Step 1 — Determine the subintervals and their lengths ($\Delta x_k$)**
The partition $P = \{1, 3, 4, 5\}$ gives three subintervals:
- $[1, 3] \implies \Delta x_1 = 3 - 1 = 2$
- $[3, 4] \implies \Delta x_2 = 4 - 3 = 1$
- $[4, 5] \implies \Delta x_3 = 5 - 4 = 1$

**Step 2 — Calculate $R_P$ using left-hand endpoints (a)**
The left endpoints for the three subintervals are $w_1 = 1$, $w_2 = 3$, and $w_3 = 4$.
$$R_P = f(1)\Delta x_1 + f(3)\Delta x_2 + f(4)\Delta x_3$$
Evaluate the function at these points:
$$f(1) = 2(1) + 3 = 5$$
$$f(3) = 2(3) + 3 = 9$$
$$f(4) = 2(4) + 3 = 11$$
Compute the sum:
$$R_P = (5)(2) + (9)(1) + (11)(1) = 10 + 9 + 11 = 30$$

**Step 3 — Calculate $R_P$ using right-hand endpoints (b)**
The right endpoints for the three subintervals are $w_1 = 3$, $w_2 = 4$, and $w_3 = 5$.
$$R_P = f(3)\Delta x_1 + f(4)\Delta x_2 + f(5)\Delta x_3$$
Evaluate the function at $x = 5$ (we already know $f(3)$ and $f(4)$):
$$f(5) = 2(5) + 3 = 13$$
Compute the sum:
$$R_P = (9)(2) + (11)(1) + (13)(1) = 18 + 11 + 13 = 42$$

**Step 4 — Calculate $R_P$ using midpoints (c)**
The midpoints for the three subintervals are $w_1 = 2$, $w_2 = 3.5$, and $w_3 = 4.5$.
$$R_P = f(2)\Delta x_1 + f(3.5)\Delta x_2 + f(4.5)\Delta x_3$$
Evaluate the function at these midpoints:
$$f(2) = 2(2) + 3 = 7$$
$$f(3.5) = 2(3.5) + 3 = 10$$
$$f(4.5) = 2(4.5) + 3 = 12$$
Compute the sum:
$$R_P = (7)(2) + (10)(1) + (12)(1) = 14 + 10 + 12 = 36$$

---

## Answer

(a) $30$
(b) $42$
(c) $36$
