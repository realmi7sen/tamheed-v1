## Problem
Evaluate $\displaystyle\int \frac{x + 34}{(x - 6)(x + 2)} \, dx$.

## Rule Used
**Partial fraction decomposition — distinct linear factors:**
$$\frac{x + 34}{(x - 6)(x + 2)} = \frac{A}{x - 6} + \frac{B}{x + 2}$$

## Solution
**Step 1 — Clear denominators.**
$$x + 34 = A(x + 2) + B(x - 6)$$

**Step 2 — Convenient values.**
- $x = 6$: $\;40 = 8A \Rightarrow A = 5$
- $x = -2$: $\;32 = -8B \Rightarrow B = -4$

**Step 3 — Integrate.**
$$\int \left(\frac{5}{x - 6} - \frac{4}{x + 2}\right) dx = 5\ln|x - 6| - 4\ln|x + 2| + C$$

## Answer
$$\int \frac{x + 34}{(x - 6)(x + 2)} \, dx = 5\ln|x - 6| - 4\ln|x + 2| + C$$

## Check by Differentiation
$$\frac{5}{x-6} - \frac{4}{x+2} = \frac{5(x+2) - 4(x-6)}{(x-6)(x+2)} = \frac{x + 34}{(x-6)(x+2)} \checkmark$$
