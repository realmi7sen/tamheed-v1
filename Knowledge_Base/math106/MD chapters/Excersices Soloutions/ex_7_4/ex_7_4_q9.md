## Problem
Evaluate $\displaystyle\int \frac{5x^2 - 10x - 8}{x^3 - 4x} \, dx$.

## Rule Used
**Factor the denominator first, then partial fractions with distinct linear factors:**
$$x^3 - 4x = x(x - 2)(x + 2)$$

## Solution
**Step 1 — Factor and set up.**
$$\frac{5x^2 - 10x - 8}{x(x - 2)(x + 2)} = \frac{A}{x} + \frac{B}{x - 2} + \frac{C}{x + 2}$$

**Step 2 — Clear denominators.**
$$5x^2 - 10x - 8 = A(x^2 - 4) + Bx(x + 2) + Cx(x - 2)$$

**Step 3 — Convenient values.**
- $x = 0$: $\;-8 = -4A \Rightarrow A = 2$
- $x = 2$: $\;20 - 20 - 8 = -8 = 8B \Rightarrow B = -1$
- $x = -2$: $\;20 + 20 - 8 = 32 = 8C \Rightarrow C = 4$

**Step 4 — Integrate.**
$$\int \left(\frac{2}{x} - \frac{1}{x - 2} + \frac{4}{x + 2}\right) dx$$

## Answer
$$\int \frac{5x^2 - 10x - 8}{x^3 - 4x} \, dx = 2\ln|x| - \ln|x - 2| + 4\ln|x + 2| + C$$
