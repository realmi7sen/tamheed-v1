## Problem
Evaluate $\displaystyle\int \frac{-19x^2 + 50x - 25}{x^2(3x - 5)} \, dx$.

## Rule Used
**Partial fraction decomposition — repeated linear factor $x^2$ plus distinct factor:**
$$\frac{-19x^2 + 50x - 25}{x^2(3x - 5)} = \frac{A}{x} + \frac{B}{x^2} + \frac{C}{3x - 5}$$

## Solution
**Step 1 — Clear denominators.**
$$-19x^2 + 50x - 25 = Ax(3x - 5) + B(3x - 5) + Cx^2$$

**Step 2 — Find the constants.**
- $x = 0$: $\;-25 = -5B \Rightarrow B = 5$
- $x = \tfrac{5}{3}$: $\;-19\cdot\tfrac{25}{9} + \tfrac{250}{3} - 25 = \tfrac{-475 + 750 - 225}{9} = \tfrac{50}{9} = C\cdot\tfrac{25}{9} \Rightarrow C = 2$
- Coefficient of $x^2$: $\;-19 = 3A + C \Rightarrow 3A = -21 \Rightarrow A = -7$

**Step 3 — Integrate.**
$$\int \left(-\frac{7}{x} + \frac{5}{x^2} + \frac{2}{3x - 5}\right) dx = -7\ln|x| - \frac{5}{x} + \frac{2}{3}\ln|3x - 5| + C$$

## Answer
$$\int \frac{-19x^2 + 50x - 25}{x^2(3x - 5)} \, dx = -7\ln|x| - \frac{5}{x} + \frac{2}{3}\ln|3x - 5| + C$$
