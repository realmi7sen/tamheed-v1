## Problem
Evaluate $\displaystyle\int \frac{1}{x\sqrt{4 - x^2}} \, dx$.

## Rule Used
**Trigonometric substitution (7.4):** for $\sqrt{a^2 - x^2}$ with $a = 2$, use $x = 2\sin\theta$.

## Solution
**Step 1 — Substitute.** Let $x = 2\sin\theta$, $dx = 2\cos\theta \, d\theta$. Then
$$\sqrt{4 - x^2} = \sqrt{4 - 4\sin^2\theta} = 2\cos\theta$$

**Step 2 — Rewrite the integral.**
$$\int \frac{1}{x\sqrt{4 - x^2}} \, dx = \int \frac{2\cos\theta}{(2\sin\theta)(2\cos\theta)} \, d\theta = \frac{1}{2}\int \csc\theta \, d\theta$$

**Step 3 — Integrate.**
$$= \frac{1}{2}\ln|\csc\theta - \cot\theta| + C$$

**Step 4 — Return to $x$ via the reference triangle** (opposite $x$, hypotenuse $2$, adjacent $\sqrt{4 - x^2}$):
$$\csc\theta = \frac{2}{x}, \qquad \cot\theta = \frac{\sqrt{4 - x^2}}{x}$$

## Answer
$$\int \frac{1}{x\sqrt{4 - x^2}} \, dx = \frac{1}{2}\ln\left|\frac{2 - \sqrt{4 - x^2}}{x}\right| + C$$

## Remark
Since $(2-\sqrt{4-x^2})(2+\sqrt{4-x^2}) = x^2$, this is equivalent to $-\frac{1}{2}\ln\left|\frac{2+\sqrt{4-x^2}}{x}\right| + C$ — either form earns full marks.
