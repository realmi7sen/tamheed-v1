## Problem
Evaluate $\displaystyle\int \frac{1}{x\sqrt{9 + x^2}} \, dx$.

## Rule Used
**Trigonometric substitution (7.4):** for $\sqrt{a^2 + x^2}$ with $a = 3$, use $x = 3\tan\theta$.

## Solution
**Step 1 — Substitute.** Let $x = 3\tan\theta$, $dx = 3\sec^2\theta \, d\theta$. Then
$$\sqrt{9 + x^2} = \sqrt{9 + 9\tan^2\theta} = 3\sec\theta$$

**Step 2 — Rewrite the integral.**
$$\int \frac{3\sec^2\theta}{(3\tan\theta)(3\sec\theta)} \, d\theta = \frac{1}{3}\int \frac{\sec\theta}{\tan\theta} \, d\theta = \frac{1}{3}\int \csc\theta \, d\theta$$

**Step 3 — Integrate.**
$$= \frac{1}{3}\ln|\csc\theta - \cot\theta| + C$$

**Step 4 — Return to $x$** (opposite $x$, adjacent $3$, hypotenuse $\sqrt{9 + x^2}$):
$$\csc\theta = \frac{\sqrt{9 + x^2}}{x}, \qquad \cot\theta = \frac{3}{x}$$

## Answer
$$\int \frac{1}{x\sqrt{9 + x^2}} \, dx = \frac{1}{3}\ln\left|\frac{\sqrt{9 + x^2} - 3}{x}\right| + C$$
