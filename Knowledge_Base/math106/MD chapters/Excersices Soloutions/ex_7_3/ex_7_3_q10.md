## Problem
Evaluate $\displaystyle\int \frac{1}{\sqrt{4x^2 - 25}} \, dx$.

## Rule Used
**Trigonometric substitution (7.4)** after an inner substitution: $\sqrt{4x^2 - 25} = \sqrt{u^2 - a^2}$ with $u = 2x$, $a = 5$, then $u = 5\sec\theta$.

## Solution
**Step 1 — Inner substitution.** Let $u = 2x$, $du = 2 \, dx$:
$$\int \frac{1}{\sqrt{4x^2 - 25}} \, dx = \frac{1}{2}\int \frac{1}{\sqrt{u^2 - 25}} \, du$$

**Step 2 — Trigonometric substitution.** Let $u = 5\sec\theta$, $du = 5\sec\theta\tan\theta \, d\theta$, $\sqrt{u^2 - 25} = 5\tan\theta$:
$$\frac{1}{2}\int \frac{5\sec\theta\tan\theta}{5\tan\theta} \, d\theta = \frac{1}{2}\int \sec\theta \, d\theta = \frac{1}{2}\ln|\sec\theta + \tan\theta| + C$$

**Step 3 — Return to $x$** ($\sec\theta = \frac{2x}{5}$, $\tan\theta = \frac{\sqrt{4x^2 - 25}}{5}$):
$$= \frac{1}{2}\ln\left|\frac{2x + \sqrt{4x^2 - 25}}{5}\right| + C = \frac{1}{2}\ln\left|2x + \sqrt{4x^2 - 25}\right| + K$$

with $K = C - \tfrac{1}{2}\ln 5$.

## Answer
$$\int \frac{1}{\sqrt{4x^2 - 25}} \, dx = \frac{1}{2}\ln\left|2x + \sqrt{4x^2 - 25}\right| + C$$
