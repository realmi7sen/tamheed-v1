## Problem
Evaluate $\displaystyle\int \frac{1}{(x^2 - 6x + 34)^{3/2}} \, dx$.

## Rule Used
**Completing the square**, then **trigonometric substitution (7.4)** for the form $(u^2 + a^2)^{3/2}$ with $u = a\tan\theta$.

## Solution
**Step 1 — Complete the square.**
$$x^2 - 6x + 34 = (x - 3)^2 + 25$$
Let $u = x - 3$, $du = dx$:
$$\int \frac{1}{(u^2 + 25)^{3/2}} \, du$$

**Step 2 — Trigonometric substitution.** Let $u = 5\tan\theta$, $du = 5\sec^2\theta \, d\theta$:
$$(u^2 + 25)^{3/2} = (25\sec^2\theta)^{3/2} = 125\sec^3\theta$$
$$\int \frac{5\sec^2\theta}{125\sec^3\theta} \, d\theta = \frac{1}{25}\int \cos\theta \, d\theta = \frac{1}{25}\sin\theta + C$$

**Step 3 — Return to $u$, then $x$** (opposite $u$, adjacent $5$, hypotenuse $\sqrt{u^2 + 25}$):
$$\sin\theta = \frac{u}{\sqrt{u^2 + 25}} = \frac{x - 3}{\sqrt{x^2 - 6x + 34}}$$

## Answer
$$\int \frac{1}{(x^2 - 6x + 34)^{3/2}} \, dx = \frac{x - 3}{25\sqrt{x^2 - 6x + 34}} + C$$
