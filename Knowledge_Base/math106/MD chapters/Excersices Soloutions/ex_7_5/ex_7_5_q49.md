## Problem
Use Theorem (7.6) to evaluate $\displaystyle\int \frac{1}{1 + \sin x + \cos x} \, dx$.

## Rule Used
**Theorem (7.6) — half-angle substitution** $u = \tan\frac{x}{2}$ with all three formulas.

## Solution
**Step 1 — Rewrite the denominator.**
$$1 + \sin x + \cos x = 1 + \frac{2u}{1 + u^2} + \frac{1 - u^2}{1 + u^2} = \frac{(1 + u^2) + 2u + (1 - u^2)}{1 + u^2} = \frac{2 + 2u}{1 + u^2}$$

**Step 2 — Substitute everything.**
$$\int \frac{1 + u^2}{2 + 2u}\cdot\frac{2}{1 + u^2} \, du = \int \frac{1}{1 + u} \, du$$

**Step 3 — Integrate.**
$$= \ln|1 + u| + C$$

## Answer
$$\int \frac{1}{1 + \sin x + \cos x} \, dx = \ln\left|1 + \tan\frac{x}{2}\right| + C$$
