## Problem

Find $f'(x)$ for each expression:
(a) $e^e$
(b) $x^5$
(c) $x^{\sqrt{5}}$
(d) $(\sqrt{5})^x$
(e) $x^{(x^2)}$

---

## Rule Used

1. **Constant Rule:** Derivative of a constant is $0$.
2. **Power Rule:** $\frac{d}{dx}[x^n] = n x^{n-1}$ for constant $n$.
3. **General Exponential Rule:** $\frac{d}{dx}[a^x] = a^x \ln a$ for constant $a$.
4. **Logarithmic Differentiation:** For $y = u(x)^{v(x)}$, take $\ln$ on both sides.

---

## Solution

**(a) $f(x) = e^e$**
Since $e^e$ is a constant:
$$f'(x) = 0$$

**(b) $f(x) = x^5$**
Apply the Power Rule with $n = 5$:
$$f'(x) = 5x^4$$

**(c) $f(x) = x^{\sqrt{5}}$**
Apply the Power Rule with constant exponent $n = \sqrt{5}$:
$$f'(x) = \sqrt{5} x^{\sqrt{5} - 1}$$

**(d) $f(x) = (\sqrt{5})^x$**
Apply the Exponential Rule with constant base $a = \sqrt{5}$:
$$f'(x) = (\sqrt{5})^x \ln(\sqrt{5}) = \frac{1}{2} (\sqrt{5})^x \ln 5$$

**(e) $f(x) = x^{(x^2)}$**
Use logarithmic differentiation since both base and exponent are functions of $x$:
Let $y = x^{(x^2)}$.
$$\ln y = \ln\left(x^{(x^2)}\right) = x^2 \ln x$$
Differentiate both sides with respect to $x$ using the product rule:
$$\frac{1}{y} y' = (2x)(\ln x) + (x^2)\left(\frac{1}{x}\right) = 2x \ln x + x = x(2\ln x + 1)$$
Multiply by $y$:
$$y' = x^{(x^2)} \cdot x(2\ln x + 1) = x^{x^2 + 1}(2\ln x + 1)$$

---

## Answer

(a) $0$
(b) $5x^4$
(c) $\sqrt{5} x^{\sqrt{5} - 1}$
(d) $(\sqrt{5})^x \ln(\sqrt{5})$
(e) $x^{x^2 + 1}(2\ln x + 1)$
