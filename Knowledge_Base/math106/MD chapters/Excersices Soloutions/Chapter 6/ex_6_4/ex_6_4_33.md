## Problem

Evaluate the integral.
$$\int \frac{\cos x \sin x}{\cos^2 x - 1} \, dx$$

---

## Rule Used

**Logarithmic Integration Rule**
$$\int \frac{1}{u} \, du = \ln|u| + C$$

---

## Solution

**Step 1 — Perform substitution**
Let $u = \cos^2 x - 1$. Differentiating using the chain rule:
$$du = 2(\cos x)(-\sin x) \, dx = -2\cos x \sin x \, dx \implies \cos x \sin x \, dx = -\frac{1}{2} du$$

**Step 2 — Rewrite the integral in terms of $u$**
$$\int \frac{\cos x \sin x}{\cos^2 x - 1} \, dx = \int \frac{1}{u} \left(-\frac{1}{2} du\right) = -\frac{1}{2} \int \frac{1}{u} du$$

**Step 3 — Integrate and substitute back**
$$-\frac{1}{2} \ln|u| + C = -\frac{1}{2} \ln|\cos^2 x - 1| + C$$

**Step 4 — Simplify using trigonometric identities (Optional)**
Since $\cos^2 x - 1 = -\sin^2 x$, we have $|\cos^2 x - 1| = \sin^2 x$:
$$-\frac{1}{2} \ln(\sin^2 x) + C = -\frac{1}{2} (2 \ln|\sin x|) + C = -\ln|\sin x| + C$$

---

## Answer

$$-\ln|\sin x| + C$$

---

## Check by Differentiation

$$\frac{d}{dx}\left[ -\ln|\sin x| \right] = -\frac{\cos x}{\sin x} = -\cot x$$
Using trigonometric identities:
$$\frac{\cos x \sin x}{\cos^2 x - 1} = \frac{\cos x \sin x}{-\sin^2 x} = -\frac{\cos x}{\sin x} = -\cot x \quad \checkmark$$
